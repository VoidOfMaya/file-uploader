### core features

1. ** User Authentication **
    - users signin + signup
    - session presist (Passport + session store)
2. ** File Upload **
    - Authenticated users see a form for file upload
    - files upload(multer/cloudinary)
3. ** Folder Management **
    - users CRUD folder
    - files belong to folders or (root)
4. ** File Details & Download ** 
    - user view:(name / type / size / uploadTime)
    - user can download files
5. ** cloud Upload **
    - file content stores on the cloud
    - database stores url referance to cloud
6. ** Validations **
    - restrict file size and type
